# Vicharanashala Frontend

A stunning, AI-powered FAQ and community Q&A platform built with React, Chakra UI, and Framer Motion.

## Features

- **AI Search:** Semantic search through FAQs.
- **Community Q&A:** Ask questions, provide answers, and vote on content.
- **Gamification:** Earn reputation points and climb the leaderboard.
- **Personalized Profile:** Track your bookmarks and notifications.

## Tech Stack

- **Framework:** React 18 (Vite)
- **UI:** Chakra UI
- **Animations:** Framer Motion
- **Data Fetching:** TanStack Query (React Query)
- **State Management:** Zustand
- **Icons:** Lucide React

## Getting Started

1.  Navigate to the `frontend` directory.
2.  Install dependencies: `npm install`
3.  Start the development server: `npm run dev`
4.  Ensure the backend is running at `http://localhost:5000`.

## Architecture

- `src/components`: Shared UI components.
- `src/pages`: Route-level components.
- `src/services`: API client and services.
- `src/store`: Global state management.
- `src/theme`: Custom Chakra UI theme.
